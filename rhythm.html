Phantom Rhythm Girl

Project Manger, Game Designer, Game Programmer


About

  • Developed in Unity
  • For Touhou Pride Jam 3
  • Half hour of gameplay, with two stages and five cutscenes
  • Team of 7 people created two stages in 15 days
  • View and download on Itch.io

Roles for Project

Project Manager
Collaborated daily with cross-disciplinary team of diverse developers
Gathered requirements and scope out project
Authored documentation to streamline development

Game Designer
Authored specification document
Designed a four note rhythm game, with chords, and note elongation
Created the scenarios in the game
Charted out each stage, keeping in time with the music

Unity Developer
Implemented the rhythm game engine, charting and mid-stage dialog
Implemented visual novel engine, allowing for cutscenes, dialog, and scene transition
